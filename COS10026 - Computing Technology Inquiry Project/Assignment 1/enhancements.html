ENHANCEMENTS

Responsive Website

Creating a responsive website is one of our group's enhancements. Its definition is to design a website that can adapt and display properly on various screen devices. In addition to developing a standard web layout, we've decided to incorporate a responsive design file to improve user experience. We were taught about this concept in lecture, and with that knowledge, we applied and enhanced the code accordingly.

Rolling Effect Animation

This enhancement goes beyond the requirements of the assignment by applying animation properties: animation-delay, animation-direction, animation-duration, and more. To make it seem like a 3D animation, I used the CSS properties Perspective and Perspective-origin. The former decides how far the user is from the element, while the latter decides from which aspect (top left, top right, center center, etc.) the user is looking.

This website is where I researched about Perspective and Perspective-origin: https://developer.mozilla.org/en-US/docs/Web/CSS/perspective-origin
